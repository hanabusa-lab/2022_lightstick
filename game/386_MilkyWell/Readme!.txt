●フォントのインストール方法
コントロールパネル内の「フォント」フォルダへ入れるだけです。

●フォントの使い方
カタカナフォントは半角英字モードで“カナ打ち”してください。
「ア」は半角の「３」、「カ」は半角の「ｔ」といった具合です。

●フォントのキー配列について
フォントを使う際、文字の割り当てが分からない場合は、
アクセサリの「文字コード表」を参照しながらお使いください。

●フォントの使用規約
フォントは個人／商用関係なく自由にお使いください。
素材制作等に使用し、それを配布したりするのもＯＫです。
但し、フォントファイル自体を他メディアへ転載したい場合には
事前に作者までご相談ください。無断転載はご遠慮願います。
